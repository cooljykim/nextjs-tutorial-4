## Jay's Next.js tutorial

Created from the Nex.js [App Router course](https://nextjs.org/learn/dashboard-app/getting-started)

### Accounts

You need to sign up for the following accounts if you don't have one already:

- [Github](https://github.com/signup)
- [Vercel](https://vercel.com/signup)

### Install tools

- Node.js 20.9 or later : [link](https://nodejs.org/en)
- [pnpm](https://pnpm.io/)
  ```
  npm install -g pnpm
  ```

### Setup Database

- Follow the steps in [Chapter 6](https://nextjs.org/learn/dashboard-app/setting-up-your-database)

### Running locally

- Clone this repository
- Go to the directory
- Install the project's packages
  ```
  pnpm i
  ```
- Start the development server
  ```
  pnpm dev
  ```
